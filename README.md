# Causal_inference_SMIH_assessment2
This is an R_based project for the MSc Health Data Science module 'Statistical Modelling and Inference for Health'. This project assessed understanding and competence with statistical modelling techniques. In particular, this assignment was primarily based around causal inference models with some testing of data cleaning and survival analysis.

IMPORTANT- Please note that this dataset was made-up as part of the course module for the purpose of assessing understanding of statistical modelling and inference. The disease 'heffpox' and treatment 'milnepan' mentioned in the data and report do not actually exist either.
